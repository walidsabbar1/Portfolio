# 🚀 Portfolio - My Developer Journey

Welcome to my personal portfolio repository! This project represents more than just a collection of code—it's a living documentation of my growth, creativity, and passion for software development.

The app will launch in your browser at https://walidsabbar.vercel.app/.

## 📖 The Journey

Every developer needs a home on the web, a place to carve out their identity. This portfolio started as a simple idea: to build something that didn't just _tell_ people what I can do, but _showed_ them through experience.

The journey began with the basics of React, but quickly evolved into a quest for perfection in User Interface (UI) and User Experience (UX). I didn't want a static, boring resume. I wanted an application that felt alive.

Over countless hours, I iterated on designs:

- Refined navigation bars to be sleek and floating.
- Experimented with **Glassmorphism** to create a modern, premium feel.
- Integrated **Lottie animations** to bring static pages to life.
- Built interactive elements like a custom **IDE-like popup** to showcase code in a familiar environment.

There were challenges—fixing layout shifts, debugging `npm` errors, refining responsive layouts—but each bug fixed was a lesson learned.

## 🌟 What This Means to Me

This portfolio is my digital handshake. It stands for:

- **Continuous Learning**: It showcases not just what I knew when I started, but what I learned along the way (animations, routing, backend integration).

## 🛠️ How It Was Built

This application is built with a focus on performance, aesthetics, and modern web standards.

### Technology Stack

- **Core**: [React.js](https://reactjs.org/) (Hooks, Functional Components)
- **Routing**: React Router DOM (v6)
- **Styling**: Custom CSS3 (Flexbox, Grid, Animations, Variables) with a focus on Glassmorphism.
- **Icons**: React Icons (Fa, Si, Vsc).
- **Animations**: Lottie React for complex vector animations.
- **Backend/Data**: Supabase (for dynamic data management/Contact form).

### Key Features

- **Dynamic Navigation**: A custom, floating navigation bar with smooth transitions.
- **Interactive "About" Section**: Featuring a profile card and a downloadable CV.
- **Skills Showcase**: A visual representation of my technical stack, utilizing animations to keep it engaging.
- **Project Gallery**: A showcase of my work (like the E-Com project and Interactive Atom Presentation) with detailed views.
- **IDE Simulation**: A custom-built UI component that mimics a code editor/terminal to display technical exercises or snippets.

## 💻 Running Locally

If you'd like to explore the code or run this locally:

1.  **Clone the repository**:
    ```bash
    git clone <repository-url>
    ```
2.  **Navigate to the project folder**:
    ```bash
    cd my-app
    ```
3.  **Install dependencies**:
    ```bash
    npm install
    ```
4.  **Start the development server**:
    ```bash
    npm start
    ```
